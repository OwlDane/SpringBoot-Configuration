### **Langkah-langkah**  

1. **Clone Repositori**:  
   ```bash
   git clone https://github.com/username/belajar-springboot-config.git  
   cd belajar-springboot-config  
   ```  
2. **Build Proyek**:  
   ```bash
   mvn clean install  
   ```

   ```bash
   mvn spring-boot:run  
   ```  
4. **Jalankan Tes**:  
   ```bash
   mvn test
   ```  
